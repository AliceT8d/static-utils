## 说明
####  如何计算文件的 sha256 哈希值？
注意，哈希值是工具的哈希值。Windows 打开 PowerShell，使用 `Get-FileHash` 命令，Linux 在终端使用 `sha256sum` 命令。
#### 工具多久更新一次？
因为工具都是手动编译的，而且编译的工具版本都比较新，目前打算当 Linux 发行版更新了工具的大版本，我才会重新来编译上传。比如，目前 nano 编辑器的大版本是 9，当 Linux 发行版的 nano 大版本更新到了 10，我才会重新编译上传。
#### 为什么不把工具直接上传到GitHub仓库？
我考虑过，但单单一个 nano 就占用了 7M，我怕后面编译了其他工具，占用比较大，仓库不允许上传了。放在 release 里面也不好，文件多了，就需要找起来就很麻烦。
