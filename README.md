### Q&A
####  如何计算文件的 sha256 哈希值？
注意：哈希值是工具的哈希值。Windows：打开 PowerShell，输入 `Get-FileHash .\nano`，Linux：终端输入` sha256sum ./nano`。
#### 工具多久更新一次？
因为工具都是手动编译的，而且编译的工具版本都比较新，目前打算当 Linux 发行版更新了工具的大版本，我才会重新来编译上传。比如，目前 nano 编辑器的大版本是9，当 Linux 发行版的 nano 大版本更新到了10，我才会重新编译上传。
#### 为什么要不把工具直接上传到 GitHub 仓库？
我考虑过，但单单一个nano就占用了7M，我怕后面编译了其他工具，占用比较大，仓库不允许上传了。
