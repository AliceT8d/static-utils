## 说明
#### 如何计算文件的 sha256 哈希值？
注意，哈希值是工具的哈希值。Windows 上打开 PowerShell，使用 `Get-FileHash` 命令，Linux 在终端使用 `sha256sum` 命令。

#### 工具多久更新一次？
因为工具都是手动编译的，而且编译的工具版本都比较新，目前打算当 Linux 发行版更新了工具的大版本，我才会重新来编译上传。比如，目前 nano 编辑器的大版本是 9，当 Linux 发行版的 nano 大版本更新到了 10，我才会重新编译上传。

#### 文件是否会包含病毒？
没有。本来就是 Linux 上的工具，你看谁会我的用啊，说不定未来某天会被某个倒霉蛋搜关键字搜出来。
